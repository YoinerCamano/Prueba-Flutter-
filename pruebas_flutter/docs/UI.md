# UI

## Páginas
- `HomePage`: conexión, lectura, acciones (ZERO, guardar, historial)
- `WeighingHistoryPage`: filtros de fecha, selección múltiple, borrado
- `BlePage`: listado de dispositivos BLE

## Widgets
- `WeightCard`: visualiza peso, estado y batería
- `MeasurementHistoryWidget`: lista mediciones desde Firebase
- `DeviceTile`: item de dispositivo

## Estados de conexión
- Dialog modal centrado en estado `Connecting`
- Tarjeta de lectura en `Connected`
